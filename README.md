# Khantoke Gold System

ระบบจัดการร้านข้าวกล่อง "Khantoke Gold" — ติดตามยอดขายรายวัน บันทึกออเดอร์ลูกค้า
จัดการสต๊อกวัตถุดิบ (รวมน้ำจิ้ม) คำนวณต้นทุน/กำไรต่อเมนู และจัดการรายชื่อลูกค้า/ผู้จำหน่าย
อัปเดตข้อมูลแบบ real-time ผ่านฐานข้อมูล MySQL — ใช้งานได้ทั้งบนคอมพิวเตอร์และมือถือ

## เทคโนโลยี

- **Backend:** PHP + MySQLi (แยกไฟล์ต่อหน้า, แยกไฟล์เชื่อมต่อฐานข้อมูล, ไม่ใช้ framework)
- **Frontend:** JavaScript (vanilla) + CSS ธรรมดา — poll ข้อมูลจากเซิร์ฟเวอร์เป็นระยะเพื่อความ real-time
- **ฐานข้อมูล:** MySQL / MariaDB

## หน้าจอในแอป

1. **หน้าหลัก** — ยอดขาย/กำไร/ต้นทุนวันนี้, แจ้งเตือนสต๊อกต่ำ, เมนูขายดี, รายการออเดอร์พร้อมปุ่มจ่ายเงิน
2. **ออเดอร์** — เลือกวันที่, เลือกเมนูที่ทำวันนี้, รับออเดอร์ต่อลูกค้าแบบเพิ่ม/ลดจำนวน, ตารางสรุปลูกค้า × เมนู
3. **สต๊อก** — จัดการวัตถุดิบ (เพิ่ม/แก้ไข/ลบ) + จัดการน้ำจิ้ม (ส่วนผสมต่อรอบ, ทำรอบใหม่, แก้ไขจำนวน/ราคา)
4. **เมนู** — จัดการเมนู, ส่วนผสมต่อกล่อง, คำนวณต้นทุน/กำไรอัตโนมัติ
5. **ตั้งค่า** — จัดการรายชื่อลูกค้า/ผู้จำหน่าย

รองรับโหมดมืด/สว่าง และปรับ layout อัตโนมัติทั้งบนมือถือ (เมนูแถบล่าง) และคอมพิวเตอร์ (เมนูแถบข้าง)

## เริ่มต้นใช้งาน

### สิ่งที่ต้องมี

- PHP 7.4 ขึ้นไป พร้อม extension `mysqli` และ `mbstring` (ใช้ตัดข้อความภาษาไทยให้ถูกต้อง)
- MySQL หรือ MariaDB
- เว็บเซิร์ฟเวอร์ (Apache/Nginx) หรือใช้ PHP built-in server สำหรับทดสอบ

### ขั้นตอนติดตั้ง

1. สร้างฐานข้อมูล (**ต้องระบุ `--default-character-set=utf8mb4`** ไม่งั้นข้อความภาษาไทยจะเพี้ยน):

   ```bash
   mysql --default-character-set=utf8mb4 -u root < schema.sql
   ```

2. แก้ค่าเชื่อมต่อฐานข้อมูลใน `db.php` ให้ตรงกับเซิร์ฟเวอร์ MySQL ของคุณ:

   ```php
   const DB_HOST = '127.0.0.1';
   const DB_USER = 'root';
   const DB_PASS = '';
   const DB_NAME = 'khantoke_gold';
   ```

3. รันเว็บเซิร์ฟเวอร์ชี้มาที่โฟลเดอร์นี้ วิธีทดสอบเร็วๆ ด้วย PHP built-in server:

   ```bash
   php -S localhost:8000
   ```

4. เปิดเบราว์เซอร์ไปที่ `http://localhost:8000/index.php`

## โครงสร้างไฟล์

```
db.php                  ไฟล์เชื่อมต่อฐานข้อมูล (MySQLi)
schema.sql              สร้างฐานข้อมูล + ตาราง + ข้อมูลตัวอย่าง
includes/
  functions.php         ฟังก์ชันช่วยเหลือร่วม (วันที่ไทย, format เงิน, สถานะสต๊อก, cache-busting ฯลฯ)
  header.php, footer.php  ส่วนหัว/ท้ายหน้าที่ใช้ร่วมกันทุกหน้า
api/
  dashboard.php, orders.php, stock.php,
  recipes.php, vendors.php, settings.php     API endpoints (JSON) อ่าน/เขียนฐานข้อมูล
assets/
  style.css, app.js      สไตล์และ JS ที่ใช้ร่วมกันทุกหน้า (polling, toast, โหมดมืด)
  dashboard.js, orders.js,
  stock.js, recipes.js, settings.js          JS เฉพาะแต่ละหน้า
index.php, orders.php,
stock.php, recipes.php, settings.php         หน้าเพจแยกไฟล์ 5 หน้า
```
